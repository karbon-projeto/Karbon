O código tem como ideia principal fazer uma ideia de como funciona o sistema MES da Gerdau, além de modelar as classes do mesmo com o que nos foi passado.


A sequencia do código se da na seguinte forma: O funcionário passa os dados (nome, e-mail e id), após isso o código pergunta qual a função do funcionário na empresa, isso acontece pois não temos acesso ao banco de dados da Gerdau, posteriormente o código leva o funcionário para sua devida área (P4 ou gestor), e após isso, se a área for de P4, ele poderá registrar uma nova falha, e se for gestor o mesmo poderá pedir um relatório de todas as causas de falhas.
